# English Immerser 0.4.0

## Highlights

- 🎬 **YouTube 来源** — 粘贴 YouTube 链接即可加载音频和字幕（需安装 yt-dlp）
- 📊 **学习统计仪表盘** — 首页新增统计分页，7/30天学习时长柱状图 + 模块使用分布图
- ✍️ **写作提示库** — 66 条英文写作提示，三句话日记每句独立提示，"灵感"按钮随时换
- 🎨 **写作模块 UI 美化** — 提示卡片化、编辑器分层、文件列表优化
- 🐛 **字幕修复** — 切换音源字幕自动清空，跟随播放自动滚动高亮

## 新增功能

### YouTube 来源
- 播放器新增 YouTube tab，粘贴链接搜索
- Rust 后端通过 yt-dlp 提取音频流和可用字幕
- 支持手动字幕和自动生成字幕，一键加载到字幕面板
- 前提：需自行安装 `yt-dlp`（`pip install yt-dlp`）

### 统计仪表盘
- 首页新增「概览/统计」分页
- 周/月度学习时长 SVG 柱状图，60 分钟目标参考线
- 五模块使用分布水平条形图
- 本周总计、打卡天数、日均时长、最常使用模块摘要

### 写作提示库
- 66 条英文提示，按句子功能分三个槽位：事实/感受/思考
- 每日确定性轮换 + 每句独立「换一句」按钮
- 自由写作编辑器「灵感」按钮，灵感建议条可使用/忽略

### 写作模块 UI
- 三句日记：彩色左边框提示卡片（蓝/玫红/琥珀），历史日期 pill 标签
- 编辑器：标题正文分割线 + 居中灵感胶囊按钮
- 文件列表：左边框高亮选中、骨架加载、图标空状态
- 统计改为轻量 inline pill 替代重型卡片

## Bug 修复
- 切换音源字幕残留 → 自动清空字幕、AB 循环、偏移
- 字幕面板滚动跟踪偏移 → 修复 children 索引
- 字幕面板增加手动 × 关闭按钮
- YouTube 搜索按钮文字溢出 → whitespace-nowrap

## 技术变更
- Rust 新增 `which` crate，`fetch_youtube` / `fetch_youtube_subtitle` 命令
- 新增 `RefreshIcon` SVG 图标
- `checkin.ts` 导出 `MODULE_LABEL_MAP`，消除 HubScene 重复常量
- HubScene 拆分为 OverviewTab / StatsTab 组件（~80 行编排层）
- 测试从 98 → 124（+26 测试）

## 兼容性
- 备份 schema 版本不变（v1），0.3.0 备份可正常导入
- 数据存储结构不变，升级无需迁移
