## 修复目标

`v0.5.0` 已确认失败在 updater 签名阶段：

```text
incorrect updater private key password: Wrong password for that key
```

本次改为使用新生成的 updater signing keypair，并同步更新仓库公钥和 GitHub Actions private key secret。
